# Pixelart Apps 1

A fun personal project to render [GNOME Circle](https://circle.gnome.org) app icons in a 1990s pixelart style. The CRT filter was created in Pixquare. This batch features Apostrophe, Chess Clock, Endeavour and Fretboard among others.
