# Dirtywave M8 Visualiser

Visualiser asset for some of the tracks I released during [Weekly Beats
2022](http://weeklybeats.com). Sadly the only way to have sound driven
animation in Blender is to bake sound to curves, so I haven't done as many
visualisers as I want to. Hopefully the geometry nodes will get some more
contemporary solution to diving animation with sound.
